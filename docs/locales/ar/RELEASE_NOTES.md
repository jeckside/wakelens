# ملاحظات إصدارات WakeLens

## v0.3.1

- تم إصلاح إجراء `إعادة التشغيل كمسؤول` عند تشغيل WakeLens من مدخل Electron للتطوير.
- تم تحسين تسمية إجراء المسؤول باللغة الروسية.

## v0.3.0

- ترجمة كاملة للواجهة والتشخيصات والتوصيات وتقارير Markdown.
- اللغات: الإنجليزية، الصينية المبسطة، الهندية، الإسبانية، العربية، الروسية.
- اختيار لغة محفوظ.
- دعم RTL للعربية.
- وثائق مترجمة.

## v0.2.0

- حالات تشخيص واضحة، إرشادات صلاحيات المسؤول، أجهزة wake-armed، أحداث Power-Troubleshooter ولوحة جديدة.
