# دليل مستخدم WakeLens

## افحص بعد حدوث المشكلة

WakeLens يكون أكثر فائدة مباشرة بعد استيقاظ غير متوقع. افتح التطبيق واختر `افحص الآن`.

إذا ظهرت رسالة صلاحيات المسؤول، اختر `إعادة التشغيل كمسؤول` ثم افحص مرة أخرى.

## قراءة النتيجة

- `عالية`: سمّى Windows مصدرًا محددًا.
- `متوسطة`: توجد إشارة قوية لكن التأكيد محدود.
- `منخفضة`: توجد أدلة مرشحة مثل الأجهزة القادرة على الإيقاظ.
- `غير معروفة`: لم يكشف Windows أدلة كافية.

## السجل والتقارير

يساعد السجل في رؤية المشتبهات المتكررة. Markdown مناسب للدعم والمنتديات وGitHub issues. JSON مناسب للتحليل التقني.

## السلامة

WakeLens لا يغير إعدادات الطاقة تلقائيًا. راجع الأدلة قبل تعطيل الأجهزة أو المهام أو المؤقتات.
