# Маркетинг WakeLens

## Слоган

Хватит гадать, почему Windows проснулась.

## Описание

WakeLens собирает разрозненные данные Windows о пробуждении ПК и превращает их в понятный диагноз, рекомендации и отчёт.

## Аудитория

- пользователи, чей ПК просыпается ночью;
- владельцы ноутбуков с разрядкой после сна;
- технические помощники, которым нужен понятный отчёт;
- опытные пользователи, уставшие вручную запускать `powercfg`.
