# Руководство пользователя WakeLens

## Сканируйте после проблемы

WakeLens наиболее полезен сразу после неожиданного пробуждения ПК. Откройте приложение и нажмите `Сканировать`.

Если приложение сообщает, что нужны права администратора, нажмите `Перезапустить от администратора` и выполните скан снова. Это не меняет настройки питания, а только позволяет собрать больше данных.

## Как читать результат

- `Высокая` уверенность: Windows назвала конкретный источник.
- `Средняя`: есть сильный сигнал, но не хватает подтверждения.
- `Низкая`: найдены кандидаты, например устройства с правом пробуждать ПК.
- `Неизвестно`: Windows не раскрыла достаточно данных.

## История и отчёты

История помогает увидеть повторяющиеся подозрения. Markdown-отчёт подходит для форумов, поддержки и GitHub issues. JSON полезен для технического анализа.

## Безопасность

WakeLens не отключает устройства, задания или таймеры автоматически. Меняйте настройки только после проверки причины.
