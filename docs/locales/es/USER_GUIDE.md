# Guía de usuario de WakeLens

## Escanea después del problema

WakeLens es más útil justo después de una activación inesperada. Abre la app y pulsa `Escanear ahora`.

Si WakeLens indica que se necesitan permisos de administrador, usa `Reiniciar como administrador` y escanea otra vez.

## Leer el resultado

- `Alta`: Windows identificó una fuente concreta.
- `Media`: hay una señal fuerte, pero falta corroboración.
- `Baja`: hay candidatos, como dispositivos con permiso de activación.
- `Desconocida`: Windows no expuso evidencia suficiente.

## Historial e informes

El historial ayuda a detectar sospechosos repetidos. Markdown es útil para soporte, foros y GitHub issues. JSON sirve para análisis técnico.

## Seguridad

WakeLens no cambia ajustes de energía automáticamente. Revisa la evidencia antes de desactivar dispositivos, tareas o temporizadores.
