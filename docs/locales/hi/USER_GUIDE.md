# WakeLens उपयोगकर्ता गाइड

## समस्या के बाद स्कैन करें

WakeLens सबसे उपयोगी तब है जब पीसी अभी-अभी अप्रत्याशित रूप से जागा हो। ऐप खोलें और `अभी स्कैन करें` चुनें।

अगर प्रशासक अनुमति की चेतावनी दिखे, तो `प्रशासक के रूप में फिर शुरू करें` चुनें और फिर स्कैन करें।

## परिणाम कैसे पढ़ें

- `उच्च`: Windows ने स्पष्ट स्रोत बताया।
- `मध्यम`: मजबूत संकेत है, पर पुष्टि सीमित है।
- `कम`: संभावित प्रमाण मिला है, जैसे जगाने में सक्षम उपकरण।
- `अज्ञात`: Windows ने पर्याप्त प्रमाण नहीं दिखाया।

## इतिहास और रिपोर्ट

इतिहास बार-बार दिखने वाले संदिग्ध दिखाता है। Markdown सहायता, फोरम और GitHub issues के लिए सुविधाजनक है। JSON तकनीकी विश्लेषण के लिए उपयोगी है।

## सुरक्षा

WakeLens पावर सेटिंग अपने आप नहीं बदलता। उपकरण, कार्य या टाइमर बंद करने से पहले प्रमाण जाँचें।
