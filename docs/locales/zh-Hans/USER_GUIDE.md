# WakeLens 用户指南

## 在问题发生后扫描

WakeLens 在电脑刚刚意外唤醒后最有用。打开应用并点击 `立即扫描`。

如果应用提示需要管理员权限，请选择 `以管理员身份重启`，然后再次扫描。

## 阅读结果

- `高`：Windows 指出了明确来源。
- `中`：有强信号，但缺少更多佐证。
- `低`：发现候选线索，例如可唤醒设备。
- `未知`：Windows 没有暴露足够证据。

## 历史与报告

历史记录帮助发现重复嫌疑项。Markdown 适合支持、论坛和 GitHub issues。JSON 适合技术分析。

## 安全

WakeLens 不会自动修改电源设置。请在禁用设备、任务或定时器前先确认原因。
